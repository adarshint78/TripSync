# 🌍 TripSync Frontend – Travel Planning UI

This is the frontend of **TripSync**, a smart travel planning platform that helps users discover, plan, and explore trips based on their preferences.

Built with modern UI/UX principles, this frontend focuses on delivering a seamless and visually engaging travel experience.

---

## 🚀 Features

### 🧭 Travel Planner UI
- Input destination, budget, days, and weather preference  
- Clean and intuitive search interface  

### 🌄 Landing Page
- Travel-themed hero section with nature/mountain visuals  
- Inspirational travel quotes  

### 📦 Travel Packages View
- Display curated travel packages  
- Filter and sort UI  

### 🗓️ Itinerary Display
- Day-wise trip breakdown UI  
- Structured and easy-to-read layout  

### 🌦️ Weather Insights
- UI for showing best time to visit  

### 🎯 Offbeat Experiences
- Cards for unique/local activities  

### ✈️ Ticket Booking UI
- Search interface for flights, trains, buses  

### 🌐 Virtual Preview Section
- Destination preview with images/videos  

### 🤝 Solo Traveler Connect
- UI for matching and connecting travelers  

---

## 🛠️ Tech Stack

- React (Vite)
- Tailwind CSS
- Zustand (state management)
- Axios (API integration)
